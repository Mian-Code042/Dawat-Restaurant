# 🍽️ Dawat – Full Stack Food Delivery Platform

## Overview

Dawat is a full-stack food delivery web application developed to provide customers with a seamless online food ordering experience. The platform allows users to browse food categories, view product details, manage their cart, place orders, and track deliveries through a modern and responsive interface.

The project was built as a practical software engineering and web development project to demonstrate full-stack application development, database integration, authentication, API development, responsive UI design, and deployment workflows.
